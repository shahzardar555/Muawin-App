# Muawin

Your Daily Household Helper — splash screen.

## Run the app

1. Ensure [Flutter](https://flutter.dev) is installed and on your PATH.
2. From this directory run:
   ```bash
   flutter pub get
   flutter run
   ```
3. Use a connected device or emulator, or run `flutter run -d chrome` for web.

## What's included

- **Splash screen**: Pastel green gradient background, "Muawin" in Poppins (center), "Your Daily Household Helper" at the bottom, and three animated loading dots.
